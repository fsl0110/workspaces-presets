# eslint-config-react-typescript

A shareable ESLint-Preset for React with TypeScript

## Installation

`yarn add --dev eslint-config-react-typescript`

`npm install --save-dev eslint-config-react-typescript`

## Configuration

### package.json

```json
"eslintConfig": {
    "extends": "react-typescript",
    "rules": {}
},
```

### .eslintrc.json

```json
{
    "extends": "react-typescript",
    "rules": {}
},
```

### .eslintrc.js

```js
{
    "extends": "react-typescript",
    "rules": {}
},
```
