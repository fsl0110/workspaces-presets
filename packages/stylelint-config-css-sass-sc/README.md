# stylelint-config-css-sass

A shareable Stylelint-Preset for Css and Sass

## Configuration

### package.json

```json
"stylelint": {
    "extends": "stylelint-config-css-sass",
    "rules": {}
},
```

### .stylelintrc

```json
{
    "extends": "stylelint-config-css-sass",
    "rules": {}
},
```

### stylelint.config.js

```js
{
    "extends": "stylelint-config-css-sass",
    "rules": {}
},
```
